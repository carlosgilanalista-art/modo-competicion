# Simulador Fase Previa UEFA 2026/27

Proyecto React (Vite) listo para desplegar. Incluye Champions League, Europa League
y Conference League en una sola app con pestañas, con los datos fluyendo en directo
entre competiciones (sin backend, sin base de datos).

## Probarlo en tu ordenador
```
npm install
npm run dev
```

## Desplegarlo (recomendado: Vercel)
1. Sube esta carpeta a un repositorio de GitHub
2. Entra en vercel.com, "Add New Project", conecta el repositorio
3. Vercel detecta Vite automáticamente — dale a "Deploy"
4. En 1-2 minutos tienes una URL pública (ej. simulador-uefa.vercel.app)

## Alternativa aún más rápida: Netlify Drop
1. Ejecuta `npm run build` (genera la carpeta `dist/`)
2. Ve a https://app.netlify.com/drop
3. Arrastra la carpeta `dist/` — te da una URL pública al instante, sin cuenta ni repositorio
